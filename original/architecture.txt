- SwiftUI
- MVVM pattern

Backend:
- Drupal 11 (JSON:API + custom endpoints)

- User

State:

- VisitState

- Cached lists + threads
- Expiration rules

Notifications:
- APNs
- Grouped delivery
- User preference filtering

- Keychain (auth)

Audio:
- AVFoundation
- Notification sounds (Mouse, Apple Crunch)

Accessibility:
- Full compliance with Web Content Accessibility Guidelines (WCAG) 2.1
- VoiceOver optimized structure
- Rotor actions
- Minimal UI complexity
